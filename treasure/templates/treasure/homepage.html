Treasure

This server implements an API for a simple card game. You and an opponent are competing for thirteen treasures. For each treasure, you and the opponent each play a card. Whoever plays the higher card wins the treasure. However, a 1 beats a 13. Using this server, you can create a player account, join games, and play games.

Available routes:

  • /players/new/NAME
  • /players/PID
  • /players/PID/games/new
  • /players/PID/games/join
  • /players/PID/games/resume
  • /players/PID/games/GID/join
  • /players/PID/games/GID
  • /players/PID/games/GID/autoplay
  • /players/PID/games/GID/play/CARD
